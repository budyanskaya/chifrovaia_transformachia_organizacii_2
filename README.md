# Цифровая трансформация организации
## Задание 2. 
Используя сервис Курсор или другой сервис по вайб-программированию, создайте ипотечный калькулятор на Python с веб-интерфейсом и с расширенным функционалом. Пользователь должен вводить сумму кредита, срок (в годах) и процентную ставку. Скрипт должен рассчитывать сумму переплаты. В результатах калькулятора должны быть отражены:   итоговая сумма кредита; переплата по процентам; примерный необходимый ежемесячный доход, который позволит обслуживать этот платёж.
Стиль внешнего вида сайта - по выбору студента

**Формат сдачи:**
- интерфейс компьютера с созданным ипотечным калькулятором
-  фрагмент архива с файлом с зависимостями .txt
